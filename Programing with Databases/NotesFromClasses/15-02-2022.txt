Junção Externa (+)

- Completa(Íntegral)

```sql
SELECT p.nm_produto, tp.nm_tipo_produto
FROM tb_tipo_produto tp
INNER JOIN tb_produto p ON (tp.id_tipo_produto = p.id_tipo_produto)
WHERE p.id_produto = 3;
```

```sql
SELECT p.nm_produto, tp.nm_tipo_produto 
FROM tb_produtos p, tb_tipos_produtos tp
WHERE p.id_tipo_produtos = tp.id_tipo_produto
AND p.id_produto = 3;
```

Junção Externa → Usada para preservar tuplas de uma determinada tabela, selecionando um externa como  direita ou esquerda ou o total, sendo que isso ira trazer os dados preservados pelo lado decidido, para preservar da esquerda se coloca na direita;
